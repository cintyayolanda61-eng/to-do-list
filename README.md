# ✅ ZenTask - To-Do List

## Deskripsi

Proyek ini merupakan website To-Do List yang dibuat menggunakan HTML, CSS, dan JavaScript. Website ini digunakan untuk membantu pengguna mencatat dan mengelola daftar tugas sehari-hari. Selain itu, website ini dilengkapi dengan berbagai fitur seperti pencarian, filter, kalender, dark mode, serta penyimpanan data menggunakan Local Storage.

## Fitur

* Menambahkan tugas
* Mengedit tugas
* Menghapus tugas
* Menandai tugas selesai
* Pencarian tugas
* Filter kategori
* Filter prioritas
* Pengurutan berdasarkan deadline
* Drag and Drop
* Pin dan Favorit
* Progress penyelesaian tugas
* Kalender mini
* Jam digital
* Dark Mode dan Light Mode
* Penyimpanan data menggunakan Local Storage
* Tampilan responsif

## Teknologi yang Digunakan

* HTML
* CSS
* JavaScript
* Local Storage
* Font Awesome
* Google Fonts
